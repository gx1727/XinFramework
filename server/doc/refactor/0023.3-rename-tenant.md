## 3. 实施步骤回顾

### 3.1 Go 包重命名

#### 3.1.1 `apps/rbac/*` → `apps/tenant/*`（6 个子包）

```bash
git mv apps/rbac/user           apps/tenant/user
git mv apps/rbac/role           apps/tenant/role
git mv apps/rbac/menu           apps/tenant/menu
git mv apps/rbac/organization   apps/tenant/organization
git mv apps/rbac/permission     apps/tenant/permission
git mv apps/rbac/resource       apps/tenant/resource
rmdir apps/rbac
```

#### 3.1.2 `framework/pkg/rbac` → `framework/pkg/tenant/auth`

```bash
git mv framework/pkg/rbac framework/pkg/tenant/auth
```

#### 3.1.3 `cmd/xin/main.go` 注册改名

import 路径全部改到 `apps/tenant/*`。

### 3.2 SQL 表重命名

```sql
ALTER TABLE user_roles        RENAME TO tenant_user_roles;
ALTER TABLE role_menus        RENAME TO tenant_role_menus;
ALTER TABLE role_resources    RENAME TO tenant_role_resources;
ALTER TABLE role_data_scopes  RENAME TO tenant_role_data_scopes;
ALTER TABLE users             RENAME TO tenant_users;
ALTER TABLE roles             RENAME TO tenant_roles;
ALTER TABLE organizations     RENAME TO tenant_organizations;
```

`menus → tenant_menus` 和 `resources → tenant_permissions` 在后续阶段完成。
