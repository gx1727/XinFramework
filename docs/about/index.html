关于 xin

xin 是一个轻量的 Go SaaS 基础框架，致力于为开发者提供一套简洁、高效、可扩展的后端开发方案。

愿景

不做臃肿的框架，不依赖复杂的 ORM。用最直接的方式掌控 SQL，用最朴素的设计构建 SaaS。

我们相信：好的框架应该让开发者专注于业务，而不是被框架本身束缚。

设计原则

简单至上

不引入不必要的复杂性。每个功能都有明确的用途。

性能优先

直接使用 pgx，无 ORM 中间层，高性能从底层开始。

可扩展

插件架构让业务模块可以热插拔，按需启用。

可维护

清晰的代码结构，完整的文档，降低维护成本。

核心技术

领域 技术 说明
Web 框架 Gin 高性能 HTTP 框架
数据库驱动 pgx/v5 PostgreSQL 高性能驱动
缓存 go-redis Redis 客户端
认证 golang-jwt Token 生成与验证
密码加密 Argon2id 业界领先的密码哈希算法

框架特性

  • 配置管理 — YAML + 环境变量覆盖
  • 多租户 — 完整的租户隔离中间件，支持 RLS
  • 日志系统 — 按天分割，自动归档
  • 服务管理 — start/stop/restart/reload 命令
  • 优雅关闭 — 信号处理，资源释放
  • 跨平台 — Windows / Linux / macOS 兼容
